# Mailer_send
Commands: 
Usage: /email (send mail) (subject) (message)

Usage:
Add it to plugins and reboot the server.

After rebooting, enter your account information (email, password, or app password) in config.yml under plugin_data\PHPMailer.
For Google, the server host is smtp.google.com.
Or, you can use AI assistance.

Once the configuration is complete, you can edit index.html.
This can also be done with AI assistance.
